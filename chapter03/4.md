### 一、把数据模型在`xadmin`中显示

* 1、在`app`下创建一个`adminx.py`的文件
* 2、类似前面介绍的`admin.py`的一样的注册

  ```py
  import xadmin

  from .models import *

  class GoogModelConfig(object):
      pass

  xadmin.site.register(GoodsModel, GoogModelConfig)
  # 这个地方没装饰器的使用方式
  ```

* 3、关于显示表名及`app`名与`admin.py`一样的使用
